# Trademarks & Brand Policy

> **Not legal advice.** This sets the project's default posture. A lawyer reviews the trademark / trade-dress position before any public launch and before shipping any vendor-branded preset.

Typecaast renders recognizable third-party UIs, which raises trademark, trade-dress, and platform-terms questions. Our posture:

## Affiliation

Typecaast is an **independent, unaffiliated tool**. Presets are described as _"&lt;Platform&gt;-style"_ or _"inspired by"_ — never "official" or "endorsed." This disclaimer stands in the README, docs, and site footer:

> Trademarks belong to their owners. Typecaast is not affiliated with or endorsed by them.

## No bundled proprietary marks (by default)

Built-in skins reproduce **layout, typography, spacing, and color** (trade dress) but do **not** ship third-party **logos, brand icons, or proprietary / licensed fonts** in this repository. Those are referenced as user-supplied assets or substituted with neutral placeholders; each skin documents what the user must provide. Only open-licensed fonts (OFL etc.) are bundled — see `PLAN.md` §19.

## Built-in vs community skins

- **Built-in skins** are vetted against this policy and have a named owner.
- **Community skins** pass a submission checklist (no bundled marks/fonts, correct "-style" naming, declared provenance, clean sample content) before listing. The registry is a directory, **not** an endorsement, and can delist non-compliant skins.
- Official vs community skins are visually distinct in the gallery.

## Capture is user-driven and user-owned

The extension and import tools capture whatever the **user** points them at; responsibility for the rights to captured UIs rests with the user. The tools surface this with a one-time in-product acknowledgement and never auto-publish captures.

## Reference screenshots

Ground-truth screenshots used for visual regression and docs carry documented provenance, are used under fair-use / nominative-use for interoperability and testing, are kept minimal, are not redistributed as standalone assets, and are excluded from published npm packages (test-only).

## Takedown

To request delisting or report a trademark/IP concern, email **trademarks@typecaast.com**. Target first response: **72 hours**. We maintain a fast path to delist any built-in or community skin on a valid complaint.

## The Typecaast name and logo

The Typecaast **name and logo** are protected regardless of code license. A fork may use the code under its tier's terms (see [`LICENSING.md`](./LICENSING.md)) but not the Typecaast brand.
